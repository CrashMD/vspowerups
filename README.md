See [releases](https://github.com/jbchouinard/vspowerups/releases).

## Build from source

### Requirements

- Python 3.7+
- pip
- Cython
- pyinstaller

### Linux

```
make
./dist/vsoptimizer
```

### Windows

```
.\build.bat
.\dist\vsoptimizer.exe
```
