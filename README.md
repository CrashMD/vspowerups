# Vampire Survivors Power Up Optimizer

![Screenshot of the game](screenshot.png?raw=true "Vampire Survivors Power Up Optimizer")

## Usage

Download vsoptimizer.py and run with Python 3.7+.


### Linux

```
make
./dist/vsoptimizer
```

### Windows

```
.\build.bat
.\dist\vsoptimizer.exe
```
